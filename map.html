To add this map to your site, follow the instructions carefully.

1. Include the following in the head of your HTML page.

The following contains script and stylesheet for displaying customized map.

Copy Code Snippet
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/sabaldahal/sabaldahal.github.io@main/user_style.css">
<script src="https://cdn.jsdelivr.net/gh/sabaldahal/sabaldahal.github.io@main/user_loadMap.js"></script>

The following contains links to OpenLayers package. Additional information can be found here.

Copy Code Snippet
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/openlayers/openlayers.github.io@master/en/v6.15.1/css/ol.css">
<script src="https://cdn.jsdelivr.net/gh/openlayers/openlayers.github.io@master/en/v6.15.1/build/ol.js"></script>

2. Download the JSON file containing data for loading map.

The following file should be included in the same root directory as your HTML files.

Click here to download JSON file
